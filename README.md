Project-Babylon
#Languages:
	currently only french is available for the story points

#Controls
	[A]/[l_arrow]: run left
	[D]/[r_arrow]: run right
	[W]/[u_arrow]: jump
	[E]: interract with object
	[#]: toggle player invincible status
	[Esc]: open pause menu
